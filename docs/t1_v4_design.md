# T1 v4 蓄势隔夜策略设计

## 一、v3 失败根因

### 1.1 数据对比
| 样本 | 交易 | 胜率 | 夏普 | 回撤 |
|------|------|------|------|------|
| 500股 | 63笔 | 66.7% | 3.70 | -4.36% |
| 1000股 | 102笔 | 56.9% | 3.06 | -7.73% |
| 2000股 | 217笔 | **45.6%** | 1.35 | -24.59% |

### 1.2 五大根因

**1. 子策略本质是"追涨"**
- 4个子策略都要求当天涨2-7%后买入
- 买入时已是高位，次日大概率面临获利回吐
- 小样本中碰到强势股有效，大样本下必然回归均值

**2. 卖出引擎用日线模拟分时，严重失真**
- 89%交易走到"30分钟超时"退出
- 止盈5%/止损-3%几乎不触发
- `(open*2 + high + low) / 4` 凭空构造均价

**3. 大盘过滤粒度太粗**
- 日线评分无法反映盘中变化
- 2025-09月评分>=50仍有31%胜率
- 2025-10月评分>=50仍有25%胜率

**4. "排除法"选股而非"筛选法"**
- 11个过滤条件只能排除坏的
- 通过排除后的股票池仍然很大
- 无法主动筛选出高概率标的

**5. 小样本过拟合**
- 500股采样 + 多轮参数调优 = 对特定样本过拟合
- random.seed(42) 恰好选中好股票

### 1.3 核心矛盾

**隔夜策略不应追涨，应找"蓄势待发"的股票**：当天还没涨，但次日大概率高开。

## 二、v4 新策略设计

### 2.1 核心理念
| 维度 | v3（失败） | v4（新设计） |
|------|-----------|------------|
| 买入逻辑 | 追涨（当天涨2-7%） | 蓄势（当天涨0-3%） |
| 选股方式 | 排除法 | 打分制筛选 |
| 核心指标 | 技术面单一 | 趋势+量价+位置+市场 |
| 卖出逻辑 | 日线模拟分时 | 基于开盘价直接决策 |

### 2.2 多维度评分体系（总分100分）

**维度1：趋势健康度（30分）**
- MA5 > MA10 > MA20 → +10分
- MA20近5日斜率为正 → +5分
- 价格在布林带中轨上方 → +5分
- MACD DIF > DEA → +5分
- 收盘在MA5上方 → +5分

**维度2：量价关系（25分）**
- 量比0.8-1.5（温和放量） → +10分
- 近3天量能递增 → +5分
- 当天成交量 > 5日均量*0.8 → +5分
- 下影线>上影线（多方力量） → +5分

**维度3：价格位置（25分）**
- 当天涨幅0-3%（蓄势区间） → +10分
- 上影线<0.5% → +5分
- 距前高<=5%（接近突破） → +5分
- 距MA60<=8% → +5分

**维度4：市场环境（20分）**
- 大盘当天上涨 → +8分
- 大盘MA5上升 → +4分
- 大盘RSI 40-65（非极端） → +4分
- 大盘近3日涨幅>0 → +4分

**买入条件**：总分 >= 65

### 2.3 卖出引擎（简化版）

基于开盘价直接决策：
1. **高开止盈**：开盘涨>=3% → 开盘价卖出
2. **低开止损**：开盘跌>=2% → 开盘价卖出
3. **正常退出**：开盘涨-2%~3% → VWAP均价卖出 `(O+H+L+C)/4`

### 2.4 目标指标
| 指标 | 最低要求 | 理想目标 |
|------|---------|---------|
| 胜率（2000股） | >= 55% | >= 60% |
| 夏普比率 | >= 2.0 | >= 3.0 |
| 最大回撤 | < -15% | < -10% |
| 盈亏比 | >= 1.3 | >= 1.5 |

### 2.5 防过拟合设计
1. 三种随机种子（42/123/456）分别采样验证
2. 前6月训练 + 后6月验证
3. 最终2000+股全量验证
