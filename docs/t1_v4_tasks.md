# T1 v4 实施任务清单

## 阶段1：核心代码实现（可并行）

### T1: 多维度评分器 `engine/t1_v4_scorer.py`
- 实现4个评分维度（趋势30 + 量价25 + 位置25 + 市场20）
- 每个维度独立函数，输入df和索引i，输出分数
- 纯技术指标计算，不依赖外部数据

### T2: v4策略类 `engine/strategies/t1_v4.py`
- 继承BaseStrategy，调用评分器
- 依赖T1完成
- 参数：score_threshold（默认65）

### T3: 简化版卖出引擎 `engine/t1_v4_sell.py`
- 3级决策：高开止盈/低开止损/正常退出
- 与T1/T2并行开发

## 阶段2：回测验证

### T4: 回测脚本 `scripts/t1_v4_backtest.py`
- 依赖T1+T2+T3
- 1000股快速验证
- 含月度分布、卖出原因分析

### T5: 评分阈值调优
- 依赖T4
- 测试阈值55/60/65/70/75
- 找到最优阈值

### T6: 防过拟合验证
- 依赖T5
- 三种随机种子验证
- 2000股全量验证
- 前后半年分割验证

## 阶段3：迭代优化（按需）

### T7: 行业数据增强（可选）
- 拉取行业分类和行业指数
- 加入行业热度评分
- 依赖T6结果判断是否需要

### T8: 策略代码更新
- 依赖T6
- 将最优参数写入策略默认配置
- 更新文档

## 执行顺序

```
T1 ─┬─→ T2 ─→ T4 ─→ T5 ─→ T6 ─→ T7(可选) ─→ T8
T3 ─┘
```

T1和T3可以并行开发。
