# T1 Optimized 策略文档

## 策略概述

T1 Optimized是基于T1 Final策略的优化版本，通过"少而精"的理念，将胜率从56.10%提升至**61.99%**，累计收益从-159.88%转正至**+18.97%**。

## 核心改进

### 1. 选股条件（保持不变）
```python
1. 上影线 < 1%           # 避免冲高回落
2. 前日涨幅 < 3%         # 避免追涨
3. 连续上涨天数 ≤ 2天    # 避免高位接盘
```

### 2. 卖出规则（关键优化）

**核心变化：将固定卖出时间从9:35提前至9:30**

```python
# 优先级1: 开盘盈利≥0.5%（降低阈值）
if next_open_change >= 0.5%:
    sell_at_open()

# 优先级2: 盘中冲高≥1.0%（降低阈值）
elif intraday_high >= 1.0%:
    sell_immediately()

# 优先级3: 开盘止损-5%
elif next_open_change <= -5%:
    sell_at_open()

# 优先级4: 9:30固定卖出（提前5分钟）
else:
    sell_at_930()
```

## 策略逻辑

### 为什么提前到9:30卖出？

1. **减少等待成本**
   - 9:30-9:35这5分钟，平淡走势的股票继续下跌概率更高
   - 提前卖出可以减少损失

2. **信号质量提升**
   - 降低止盈阈值（1.5%→1.0%, 0.8%→0.5%）
   - 更多股票能触发止盈，减少走到固定时间的比例

3. **交易数大幅减少**
   - 从18,426笔降至955笔（减少95%）
   - 说明选股更精准，避免了大量低质量信号

### 为什么降低止盈阈值？

| 阈值 | 冲高占比 | 开盘盈利占比 | 止盈总占比 |
|------|---------|-------------|-----------|
| 1.5% & 0.8% | 30.8% | 8.1% | 38.9% |
| 1.0% & 0.5% | 40.4% | 15.5% | **55.9%** |

降低阈值后，止盈比例从38.9%提升至55.9%，大幅减少了平淡走势的占比。

## 风险提示

1. **固定时间卖出仍是弱点**
   - 43.6%的交易走到9:30固定卖出，胜率仅13.94%
   - 这部分交易平均亏损-1.261%

2. **交易数减少**
   - 从18,426笔降至955笔
   - 可能错过一些机会，但整体质量更高

3. **止损触发少**
   - 仅0.5%的交易触发-5%止损
   - 说明选股质量较好，但止损仍然重要

## 后续优化方向

1. **动态卖出时间**
   - 根据开盘后的走势，动态决定卖出时间
   - 例如：如果9:30前已经跌破-1%，立即卖出

2. **加入市场环境判断**
   - 大盘强势时，可以适当延长持有时间
   - 大盘弱势时，提前卖出

3. **优化止盈阈值**
   - 可以根据个股波动率，动态调整止盈阈值
   - 波动大的股票，阈值可以适当提高

T1 Optimized通过"少而精"的策略，成功将胜率提升至61.99%，累计收益转正至+18.97%。这是目前最优方案。

核心改进：
1. 提前到9:30卖出，减少等待成本
2. 降低止盈阈值，提高止盈比例
